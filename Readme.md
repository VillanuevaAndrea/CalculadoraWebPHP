#Calculadora Web con PHP y Bootstrap

Este es un proyecto desarrollado en **PHP**, donde se implementó una calculadora funcional que maneja peticiones mediante el método **POST** y mantiene un diseño moderno y responsivo utilizando **Bootstrap 5**.

## Características
- **Operaciones básicas:** Suma, resta, multiplicación y división.
- **Validaciones:** Control de errores para divisiones por cero y entradas no numéricas.
- **Persistencia de datos (Sticky Forms):** Los números ingresados permanecen en los campos después de realizar el cálculo para mejorar la experiencia del usuario.
- **Botón de Limpieza:** Lógica para resetear el formulario y limpiar el historial de cálculos.

## Tecnologías utilizadas
* **PHP 8.x:** Lógica del lado del servidor y manejo de variables superglobales (`$_POST`).
* **Bootstrap 5:** Maquetación rápida y diseño responsivo sin necesidad de CSS manual complejo.
* **XAMPP / Apache:** Entorno de servidor local para la ejecución del código.
* **Git/GitHub:** Control de versiones.
